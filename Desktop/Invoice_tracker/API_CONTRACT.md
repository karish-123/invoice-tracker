### CHECKOUTS â€” ADMIN + OFFICE_STAFF

#### `POST /checkouts/issue`
```json
// Request
{
  "executiveId": "uuid",
  "routeId": "uuid",
  "outDatetime"?: "ISO8601 datetime string (optional, defaults to now)",
  "invoiceNumbers": ["string", ...]   // min 1
}

// Response 207 (mixed) or 422 (all failed)
{
  "results": [
    { "invoiceNumber": "string", "success": true,  "checkoutId": "uuid" },
    { "invoiceNumber": "string", "success": false, "error": "Invoice X is already checked out" }
  ]
}

// Error 400: { "error": "Executive not found or inactive" }
// Error 400: { "error": "Route not found or inactive" }
```

#### `POST /checkouts/return`
```json
// Request
{
  "invoiceNumbers": ["string", ...],   // min 1
  "inDatetime"?: "ISO8601 datetime string (optional, defaults to now)"
}

// Response 207 or 422 â€” same results[] shape as issue
// Per-invoice error: "Invoice X has no active checkout"
```

#### `GET /checkouts/outstanding`
Query params (all optional): `executiveId`, `routeId`, `olderThanDays`
```json
// Response 200 â€” array of Checkout:
{
  "id": "uuid",
  "invoiceNumber": "string",
  "executive": { "id": "uuid", "name": "string" },
  "route": { "id": "uuid", "routeNumber": "string" },
  "outDatetime": "ISO8601",
  "outByUser": { "id": "uuid", "name": "string" },
  "inDatetime": null,
  "inByUser": null,
  "status": "OUTSTANDING",
  "voided": false,
  "voidReason": null,
  "createdAt": "ISO8601"
}
```

#### `POST /checkouts/:id/void`
```json
// Request: { "voidReason": "string (min 1)" }
// Response 200: Checkout object with status "VOIDED"
// Error 400: "Checkout is already voided"
// Error 400: "Cannot void a returned checkout"
// Error 404: "Checkout not found"
```
